# 🌌 Acerca del proyecto

**SistemaREMI 3.0** es un sistema de ventas enfocado en la administración de tiendas de electrodomésticos, permitiendo gestionar productos, inventario, ventas y clientes desde una plataforma centralizada.

El sistema fue diseñado para facilitar:

- 🛒 Gestión de ventas
- 📦 Control de inventario
- 👤 Administración de clientes
- 💳 Registro de pagos
- 📊 Reportes comerciales
- 🏷️ Gestión de productos
- ⚡ Automatización de procesos

El proyecto fue desarrollado para practicar:

- Sistemas administrativos
- Bases de datos
- Gestión comercial
- Desarrollo web
- Arquitectura CRUD
- Interfaces administrativas

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/usuario/SistemaREMI-3.0.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd SistemaREMI-3.0
```

---

## 3️⃣ Configurar base de datos

Importar archivo SQL:

```bash
database/sistemaremi.sql
```

---

## 4️⃣ Ejecutar servidor

- Iniciar Apache
- Iniciar MySQL

---

## 5️⃣ Abrir sistema

```bash

```
